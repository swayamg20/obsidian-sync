1. Physical Layer:
    
    - Ethernet, Wi-Fi, DSL, Fiber Optic
2. Data Link Layer:
    
    - Ethernet, PPP (Point-to-Point Protocol), HDLC (High-Level Data Link Control)
3. Network Layer:
    
    - IP (Internet Protocol), ICMP (Internet Control Message Protocol), IGMP (Internet Group Management Protocol)
4. Transport Layer:
    
    - TCP (Transmission Control Protocol), UDP (User Datagram Protocol)
5. Session Layer:
    
    - NetBIOS, RPC (Remote Procedure Call)
6. Presentation Layer:
    
    - SSL/TLS (Secure Sockets Layer/Transport Layer Security), MIME (Multipurpose Internet Mail Extensions)
7.  [[Application Layer]] Application Layer:
    
    - HTTP, HTTPS, FTP, SMTP, DNS, SSH, Telnet
